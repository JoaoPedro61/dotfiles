# My dotfiles and setup script

---

## Execute installers and configuration setup
```bash
git clone https://github.com/joaopedro61/dotfiles.git && chmod +x ./dotfiles/configure.sh && ./dotfiles/configure.sh all
```


## Execute installers
```bash
git clone https://github.com/joaopedro61/dotfiles.git && chmod +x ./dotfiles/configure.sh && ./dotfiles/configure.sh packages
```


## Execute configuration setup
```bash
git clone https://github.com/joaopedro61/dotfiles.git && chmod +x ./dotfiles/configure.sh && ./dotfiles/configure.sh configs
```
